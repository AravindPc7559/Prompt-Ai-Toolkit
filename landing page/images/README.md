# Extension Images

Place your extension screenshots in this folder with the following names:

## Required Images

1. **hero-extension.png** (Recommended: 1000x600px)
   - Main hero image showing the extension popup or interface
   - Should be high quality and showcase the main features

2. **extension-popup.png** (Recommended: 800x600px)
   - Screenshot of the extension popup interface
   - Shows the main UI with input fields and buttons

3. **floating-icons.png** (Recommended: 800x600px)
   - Screenshot showing the floating icons on a webpage
   - Demonstrates the text selection and enhancement features

4. **text-selection.png** (Recommended: 800x600px)
   - Screenshot showing text selection with enhancement options
   - Shows how the extension works on actual web pages

5. **dashboard.png** (Recommended: 800x600px)
   - Screenshot of the user dashboard
   - Shows subscription management and usage tracking

## Image Guidelines

- **Format**: PNG or JPG
- **Aspect Ratio**: 4:3 or 16:9 recommended
- **Quality**: High resolution (at least 800px width)
- **File Size**: Optimize images to keep page load fast (under 500KB each)
- **Content**: Make sure screenshots are clear and showcase the features well

## Fallback

If images are not found, the page will show placeholder images with colored backgrounds and text labels.
